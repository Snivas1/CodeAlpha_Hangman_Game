# 🎮 CodeAlpha Hangman Game

A professional console-based **Hangman Game** developed in **Python** as part of the **CodeAlpha Python Programming Internship**.

The game features a clean modular architecture, multiple difficulty levels, hints, colorful terminal output, ASCII Hangman graphics, and an interactive menu-driven interface.

---

## 📌 Features

- 🎮 Interactive Main Menu
- 📖 Instructions Menu
- 🎯 Difficulty Levels
  - Easy
  - Medium
  - Hard
- 💡 Word Hints
- 🎨 Colorful Console Output
- ❤️ Lives Display
- 🔤 Letter-by-Letter Guessing
- ⚠️ Input Validation
- 🚫 Duplicate Guess Detection
- 🎭 ASCII Hangman Graphics
- 🏆 Winning & 💀 Game Over Screens
- 🔄 Replay Support
- 🧩 Modular Python Project Structure

## ▶️ How to Run

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/CodeAlpha_Hangman_Game.git
```

### Navigate to the project

```bash
cd CodeAlpha_Hangman_Game
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the game

```bash
python main.py
```
